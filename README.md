# Task 2 - Data Visualization & Storytelling

## Dataset Used
Amazon Prime Titles Dataset

## Tool Used
Power BI

## Dashboard Summary
This dashboard provides insights about the content available on Amazon Prime. It focuses on
content distribution across genres, ratings, release years, and countries.

## Visuals Included
- KPI Cards (Total Directors, Total Genres, etc.)
- Shows by Genre (Bar Chart)
- Shows by Audience Rating (Bar Chart)
- Content by Country (Map)
- Movies vs TV Shows Split (Donut Chart)
- Release Trend Over Time (Line Chart)

## Key Insights
1. Drama and Comedy are the most popular genres on Amazon Prime.
2. Most titles fall under age ratings of 13+ and 16+, showing focus towards young adult viewers.
3. USA and India contribute the highest number of shows.
4. Movies are more in quantity than TV Shows.
5. There has been a sharp rise in content releases after 2010.
